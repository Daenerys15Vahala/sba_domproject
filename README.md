SBA DOM PROJECT

Fratushka Cafe

This project was created to demonstrate JavaScript DOM and BOM manipulation and form validation.
Fratushka Cafe is a web page application that allows users to customize and place a coffee order.
Users enter their information and receive a summary with the total price of their order.

Features
-Enter customer name and email
-Select a coffee drink
-Choose drink size
-Add additional extras
-Give order summary updates
-Automatic total price calculation
-Confirmation after placing order
-Clearing our order form
